app.track

app.track is designed to provide job-seekers in software development and related fields the ability to review the application process for specific companies. Users are able to create an account, log in, write reviews, and read reviews posted by others.



Technical Features
  • Uses Express templating for site structure
  • Uses pg-promise to establish database connection, and run SQL queries such as getting company data and posting reviews
  • Secure password hashing and storage completed via bcrypt
  • Dynamic interface
  • Designed with Bulma framework

PokéFind

Pokéfind is an application that provides statistics on every Pokemon character to date. It utilizes a search bar and has the option to randomly select a character.



Technical Features
  • Uses JavaScript Fetch functionality to manage responses
  • Connects to the PokéAPI site through multiple channels
  • Utilizes jQuery for DOM manipulation
  • Designed with Bootstrap framework

Llamarita's Taco Adventure

Llamarita's Taco Adventure is a game built using the Pygame library. Players move the character around the platform in order to avoid falling cacti and collect Margaritas to win, each of which increases the number of falling cacti. Tacos are collected as bonus points.



Technical Features
  • Built with Python and utilizes the Pygame library
  • Difficulty increases as score increases
  • Features audio track and sound effects

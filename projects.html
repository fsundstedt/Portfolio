app.track

app.track is an application designed to provide job-seekers in software development and related fields the ability to review the application process for specific companies. Users are able to create an account, log in, write reviews, and read reviews posted by others.



Technical Features
  • Uses Express templating for site structure
  • Uses pg-promise to utilize database connection and run SQL queries
  • Password hashing completed via bcrypt
